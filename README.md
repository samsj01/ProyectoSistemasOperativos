##  3. Descripción Breve
El sistema es una herramienta diseñada para administrar y visualizar el ciclo de vida de los procesos en un entorno controlado. Permite simular la ejecución de tareas, gestionar sus estados y monitorear recursos mediante algoritmos de planificación.

##  4. Arquitectura de Módulos
1. Cargador (Loader): Inicialización de procesos.
2. Cola de Listos: Gestión de espera.
3. Planificador (Scheduler): Lógica de CPU.
4. Simulador de CPU: Ejecución de ráfagas.
5. Monitor de Recursos: Captura de métricas.
6. Interfaz (CLI): Visualización de logs.

## 🎯 5. Alcance de E1
* **Componente:** Motor de Transición de Estados.
* **Criterio de Aceptación:** Carga de procesos y visualización de cambios de estado completos en terminal.
